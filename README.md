# MSC INPT - Math & Science Club Website

Welcome to the official repository for the **MSC INPT** website. This project showcases the club's research, events, and a premium registration experience for future members.

## 🏛️ Project Architecture

```text
/
├── apply/              # Registration experience
│   └── index.html      # Application form page
├── pages/              # Additional website pages
│   └── research.html   # Research Hub (Scientific articles)
├── css/                # Style Definitions
│   ├── style.css       # Main website styles & Animations
│   ├── form.css        # Registration-specific glassmorphic styles
├── js/                 # Logic & Interactivity
│   ├── main.js         # ScrollTrigger, Preloader & GSAP Animations
│   └── form.js         # Application submission & Modal logic
├── images/             # Visual Assets
│   ├── logo.png        # Official MSC branding
├── index.html          # Landing Page (The Core)
└── README.md           # Project documentation
```

## 🚀 Tech Stack

- **Core**: HTML5, Vanilla CSS3, Modern JavaScript (ES6+)
- **Animations**: GSAP (GreenSock Animation Platform), ScrollTrigger
- **Styling**: Bootstrap 5 (Grid), Glassmorphism, Swiss Grid Design
- **Icons**: Bootstrap Icons

## 🧠 Design Philosophy

The website follows a **"Premium Minimalist"** aesthetic, blending scientific rigor with modern digital design. Key features include:
- **Glassmorphism**: Sophisticated frosted surfaces for high-tech feel.
- **Editorial Typography**: Using *Playfair Display* for accents against clean *Inter* body text.
- **Dynamic UX**: GSAP-powered transitions and a bespoke preloader for a seamless load experience.
